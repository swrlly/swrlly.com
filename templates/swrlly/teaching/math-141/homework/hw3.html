Homework 3

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. Perform the indicated operation and/or fully simplify. Get rid of all negative exponents.
    1. $\dfrac{x^{-1} + y^{-1}}{4} \ \ $ (hint: use definition of negative exponent. This is a compound fraction)
    2. $\dfrac{\dfrac{1}{\sqrt{x+h}} - \dfrac{1}{\sqrt{x}}}{h} \ \ $ (hint: get rid of the compound fraction, rationalize the denominator, and cancel common factors)
    3. $\dfrac{1 + \dfrac{1}{x}}{\dfrac{1}{x} - 2}$
    4. $\dfrac{\frac{1}{2x+1}}{\dfrac{1}{2x^2 + 5x + 2}}$
  2. Draw a coordinate plane and graph the points $(1,2), (2, -1), (-3, -4)$ and $(4, -3)$.
  3. Suppose I have an expression $f(x)$ and I find two different inputs that give the same evaluation. In particular, I find that $x = 2$ and $x = 3$ gives $f(2) = f(3)$. Is $f$ a function?
  4. Suppose I have an expression $f(x)$ and I find one input which gives two different evaluations. In particular, I find that $x = 2$ spits out $f(2) = 5$ and $f(2) = 3$. Is $f$ a function?
  5. Write down two examples of functions which you encounter in your daily life.
  6. Let $f(x) = x^2 - x + 1$. Evaluate and simplify the following:
    1. $f(1)$
    2. $f(a)$
    3. $f(-a)$
    4. $f(x + h)$
    5. $f(x + h) - f(x)$
    6. $f(-x^2)$
  7. Let $f(x) = \dfrac{x + 1}{x^2 + x}$ Evaluate and simplify the following:
    1. $f(-1)$
    2. $f(a + h)$
    3. $f(a + h) - f(x)$
    4. $f(x^2)$
  8. Let \[f(x) = \begin{cases} 3x + 2 & x \geq 2 \\ x^2 - x & x < 2\end{cases}\] Evaluate the following:
    1. $f(0)$
    2. $f(1)$
    3. $f(2)$
    4. $f(3)$
  9. Draw a rough sketch of the previous graph by hand. You may verify your graph in Desmos.
    Hint: To graph a piecewise function in Desmos, you need to specify a domain restriction. You can graph $3x + 2, x \geq 2$ by typing the symbols $3x + 2 \{x \geq 2\}$ into Desmos, where the $\geq$ sign is created by first typing $>$ then $=$.
  10. Read these two lecture notes on trigonometric functions: 5.1: Unit Circle and 5.2: Trigonometric Functions. This is mandatory reading if you do not remember trigonometric functions.
    Write "Without using the internet, I can evaluate trigonometric functions and know their graph shapes." to get credit for this problem.
  11. For what inputs $x$ does the function $f(x) = \cos(x)$ output $0$? (Hint: 5.1 + 5.2 Lecture notes. In other words, which radian value results in an $x$-coordinate of 0 on the unit circle?).
  12. Find the six trigonometric functions of $t = \frac{2\pi}{3}$.
  13. Find the six trigonometric functions of $t = -\frac{5\pi}{6}$.
  14. If $\sin t > 0$ and $ \cos t < 0$, what quadrant is the terminal point in?
  15. Suppose $f(x) = 8^x$. Evaluate $f\left(\dfrac{2}{3}\right)$ by using the definition of the fractional exponent.
  16. You are given a function and a input value. Evaluate the function on the input value and make sure intermediate steps which show your thought process:
    1. $f(x) = \log_3 x, \qquad x = 27$
    2. $f(x) = 2^x + 4^x, \qquad x = 2$
    3. $g(x) = \tan(x), \qquad x = \dfrac{\pi}{3}$
    4. $h(x) = \dfrac{e^x + e^{-x}}{2}, \qquad x = 0$
    5. $f(x) = e^{e^x}, \qquad x = 0$
  17. Suppose \[f(x) = \sin(x) + \cos(x) \qquad g(x) = e^\frac{x}{\pi} - 1\] Evaluate and simplify the following expressions without a calculator/the internet:
    1. $f(0)$
    2. $g(0)$
    3. $f\left(\frac{\pi}{2}\right) + g(\pi)$
    4. $f(x)\cdot g(x)$
    5. $f(x) - g(x)$
  18. Redefine the following rational functions with their domain exclusions (holes):
    1. $f(x) = \dfrac{x^2 - 1}{x-1}$
    2. $f(x) = \dfrac{2x^2 + 3x - 2}{x + 2}$
    3. $f(x) = \dfrac{x^3 - 3x^2 - 4x}{x}$
  19. Suppose \[f(x) = \sin(x) \qquad g(x) = x^2 - x \qquad h(x) = f(x)g(x)\] Evaluate the following:
    1. $f\circ g$
    2. $g \circ f$
    3. $h \circ f$
  20. You are given a function $F(x)$. Find two functions $f, g$ where $F = f \circ g$.
    1. $F(x) = \sin(\cos x)$
    2. $F(x) = \sin^2(x)$
    3. $F(x) = \cos(x^2)$
    4. $F(x) = (x^3 - x^2 - 1)^{2/3}$
    5. $F(x) = e^{x^2 - x}$
    6. $F(x) = (e^x)^3 - \sin(e^x)$
    7. $F(x) = e^{e^x}$
  21. If someone writes $f(x) = \cos x + 2$, do they mean the function $\cos(x) + 2$ or $\cos (x + 2)$? This is why parentheses are important.
  22. Find the domain of the following functions:
    1. $f(x) = \dfrac{1}{x^2 - 1}$
    2. $f(x) = \dfrac{1}{\cos x} \ $ (Hint: Problem 12).
  23. Draw a rough sketch by hand of the functions \[\ln x \qquad \sin x \qquad \cos x \qquad e^x \qquad y = mx + b, m > 0\]
  24. Describe the general graph shape in English for the following functions. The first problem is done for you.
    Note: you are expected to be able to do these without technology, for example for in-class assignments.
    1. $f(x) = mx + b$
      This function's graph is a line. This is because in the formula $f(x)$ the variable you need to substitute is $x$. Since the largest power of $x$ among the terms is one (the first term), this function's parent function is $g(x) = x$, which we know is a line from Lecture Note V.
    2. $f(x) = ax^2 + bx + c$ where $a,b,c \in \mathbb{R}$ and $a \neq 0$.
    3. $f(x) = \sin(x)$
    4. $g(x) = \sqrt{x}$
    5. $a(x) = e^x$
