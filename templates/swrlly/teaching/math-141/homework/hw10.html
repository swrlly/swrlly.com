Homework 10

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. Sketch a graph of the function which satisfies the following:
    • $f'(2) = 0, \ f(2) = -1, \ f(0) = 0$
    • $f'(x) < 0$ when $0 < x < 2$
    • $f'(x) > 0$ when $ x > 2$
    • $f''(x) < 0$ when $0 < x < 1$ or $x > 4$
    • $f''(x) > 0$ when $1 < x < 4$
    • $\displaystyle \lim_{x\rightarrow \infty} f(x) = 1$
  2. When using Rolle's Theorem or the Mean Value Theorem, what do you need to check first before using them?
  3. Verify the function \[f(x) = 2x^2 - 4x + 5 \] satisfies the three hypotheses of Rolle's Theorem on the interval $[-1, 3]$. Then find all numbers $c$ that satisfy the conclusion of Rolle's Theorem.
  4. Using English, describe the intuition behind the Mean Value Theorem.
  5. For the following functions, find the number $c$ that satisfies the conclusion of the Mean Value Theorem on the given interval.
    1. $f(x) = \sqrt{x}, \ [0, 4]$
    2. $f(x) = x^3 - 2x, \ [-2, 2]$
  6. Suppose $f(x) = 1 - x^{2/3}$. Show $f(-1) = f(1)$ but there is no $c \in (-1, 1)$ where $f'(c) = 0$. Why does this not contradict Rolle's Theorem?
    Hint: Differentiability was violated somewhere. Find where $f'(x)$ DNE.
  7. Suppose that $f$ is continuous on $[a, b]$. Must there be a number $c \in (a, b)$ where \[f'(c) = \dfrac{f(b) - f(a)}{b - a}\] If not, draw a graph in which such a $c$ cannot exist.
  8. Suppose $f(x) = x^4$. Find three different antiderivatives $F_1(x), F_2(x)$ and $F_3(x)$ of $f(x)$.
  9. Find the most general antiderivative of the following functions:
    1. $f(x) = 5$
    2. $f(x) = 4x + 7$
    3. $f(x) = 3\cos x - 4 \sin x$
    4. $f(x) = x(x-1)$
    5. $f(x) = \dfrac{10}{x^9}$
    6. $g(\theta) = -\sec^2 \theta$
    7. $f(x) = 6x^5 - 8x^4 - 9x^2$
    8. $f(x) = \sqrt{x} - 3\sqrt[4]{x^3}$
  10. Evaluate the following sums:
    1. $\displaystyle\sum^5_{i=1}i - 1$
    2. $\displaystyle\sum^5_{i=1}f(i)$ given that $f(x) = x^2$.
    3. $\displaystyle\sum^4_{n=1} n(n-1)$
  11. Approximate the area underneath the graph of $f(x) = \sin x$ on $\left[0, \frac{\pi}{2}\right]$ by using four rectangles and right endpoints.
    Only set up the sum; do not find the actual number.
  12. Approximate the area underneath the graph of $f(x) = \dfrac{1}{x}$ on $\left[1, 2\right]$ by using four rectangles and right endpoints.
    Only set up the sum; do not find the actual number.
  13. Approximate the area underneath the graph of $f(x) = x^3$ on $\left[0, 1\right]$ by using four rectangles and right endpoints.
    Only set up the sum; do not find the actual number.
  14. If I would like to find the area underneath the curve of $f(x)$ on the interval $[a, b]$, what is the exact formula for this area?
  15. In the context of finding the area underneath the curve, explain what each of the parts of the formula is describing:
    1. The $\displaystyle\lim_{n\rightarrow \infty}$
    2. The $\displaystyle\sum^{n}_{i=1}$
    3. The $f(x_i) \cdot \Delta x$
  16. If $\displaystyle \int^{5}_{-5} f(x) \ dx = 12$ and $\displaystyle \int^1_{-5} f(x) \ dx = 3$, find $\displaystyle \int^{1}_{-5} f(x) \ dx$.
  17. We saw from lecture that \[\int^1_0 x^2 \ dx = \dfrac{1}{3}\] What is $\displaystyle\int^0_1 x^2 \ dx$?
  18. If $\displaystyle \int^{8}_0 f(x) \ dx = 3$ and $\displaystyle \int^0_{-3} f(x) \ dx = 4$, what is $\displaystyle \int^8_{-3} f(x) \ dx$?
  19. If $\displaystyle \int^4_2 f(x) \ dx = 3$ and $\displaystyle \int^4_2 g(x) \ dx = -2$, evaluate the following integrals
    1. $\displaystyle \int^4_2 [f(x) + g(x)] \ dx$
    2. $\displaystyle \int^4_2 [f(x) - g(x)] \ dx$
  20. Why is the Fundamental Theorem of Calculus extremely useful?
  21. Find the derivative of the following functions:
    1. $g(x) = \displaystyle \int^x_0 \sqrt{t + t^3} \ dt$
    2. $g(x) = \displaystyle \int^x_0 \sin(t^2) \ dt$
    3. $F(x) = \displaystyle \int^0_x \sqrt{t^3} \ dt$
      Hint: Property 6 of Definite Integrals.
  22. In English, what does FTC Part 2 tell us about how to find definite integrals?
  23. Evaluate the following integrals using FTC Part 2:
    1. $\displaystyle \int^5_{-5} 3 \ dx$
    2. $\displaystyle \int^1_0x^2 \ dx$
    3. $\displaystyle \int^3_1(x^2 + 2x - 4) \ dx$
    4. $\displaystyle \int^\pi_{\pi/6} \sin \theta \ d\theta$
    5. $\displaystyle \int^1_0 (x-1)(x+1) \ dx$
    6. $\displaystyle \int^{\pi/3}_{\pi/6} \sec^2 x\ dx$
  24. Find the area enclosed between the curve and the $x$-axis of the function $y = \sqrt{x}$ on the interval $[0, 4]$.
  25. Why does the integral \[\int^1_{-1} \dfrac{1}{x} \ dx\] not exist?
