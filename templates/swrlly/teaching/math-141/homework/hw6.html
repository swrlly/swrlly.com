Answer the following:

  1. Find the derivative using the Differentiation Rules. Remember to observe your input variable and to fully simplify each expression.
    1. $f(x) = 199$
    2. $g(t) = \left(\sqrt{30}\right)^3$
    3. $f(x) = x^3 + x^2 + x + 1$
    4. $f(v) = \dfrac{a}{v} + bv + \dfrac{c}{v^2}$
    5. $f(x) = \sqrt{x}(x - 1)$
    6. $f(t) = 3t^{-3/4}$
    7. $f(x) = x^{2.9}$
    8. $f(r) = -9r^{2/3}$
    9. $f(x) = 5x^2 + 6x + 7$
    10. $f(r) = \dfrac{r^5 + 2r^4 + 3r + 1}{r}$
    11. $f(a) = \dfrac{5}{a^4} - \dfrac{2}{\sqrt[3]{a^2}} -\dfrac{1}{a} + 200$
    12. $f(x) = \dfrac{5}{x^4} + \dfrac{\sqrt{x^3}}{3} + x$
    13. $f(x) = 2x\left(3x^2 + 1\right)$
    14. $f(x) = (3x + 1)(4x^3 - 5x^2)$
    15. $f(t) = \dfrac{t - \sqrt{t}}{t^{1/3}}$
    16. $y = \dfrac{x^2 + 1}{x^2 - 1}$
    17. $y = x + \dfrac{1}{x}$
    18. $y = \dfrac{1 + 2x}{3 - 4x}$
    19. $f(x) = \dfrac{x}{x + \frac{1}{x}}$
    20. $f(x) = (x + 1)(x^{-2} + x^{-3})$
  2. Suppose a student sees a function \[f(x) = \dfrac{\sqrt{x}}{x + 1}\] and is asked to find $\dfrac{d}{dx} f(x)$. The student tries this: \[\dfrac{d}{dx} f(x) = \dfrac{d}{dx} \dfrac{\sqrt{x}}{x + 1} = \dfrac{\dfrac{d}{dx} \sqrt{x}}{\dfrac{d}{dx} (x + 1)}\] Why is this incorrect?
  3. Given $g(r) = \sqrt{r} + \sqrt[3]{r}$, find the first and second derivatives.
  4. The equation of motion of a particle is $s(t) = t^4 - 2t^3 + t^2 - t$ where $s$ is in meters and $t$ is in seconds.
    1. Find the velocity and acceleration as functions of $t$.
    2. Find the acceleration when the velocity is 0. (hint: set $v(t) = 0$ to find when the velocity is 0).
  5. Find an equation of the normal line to the curve $y = \sqrt{x}$ that is parallel to the line $2x + y = 1$.
    Hint: Find the derivative and set it equal to the needed slope. Solve for $x$, this value of $x$ gives you the $x$-value of the point of tangency.
  6. Differentiate the following:
    1. $f(x) = \sec x$
    2. $f(x) = \cot x$
    3. $f(x) = x\cos x + 2\tan x$
    4. $f(x) = \dfrac{\sec x}{1 + \tan x}$
    5. $w(t) = \dfrac{\sin t}{1 - \sin t}$
  7. Find the 51st derivative of $f(x) = \sin x$.
  8. Differentiate the following:
    1. $f(x) = (3x - x^3)^{23}$
    2. $f(x) = \sqrt[3]{1 + \tan x}$
    3. $f(x) = (3x + 2)^3(4x^2 + 3)^4$
    4. $g(x) = \cos^3 x$
    5. $g(x) = \cos x^3$
    6. $y = \sqrt{x + \sqrt{ x + \sqrt{x}}}$
    7. $f(x) = \sqrt{\dfrac{x}{x^2 + 4}}$
    8. $f(x) = \sin x + \sin^2 x$
    9. $g(a) = \sin(x \sin x)$
    10. $f(x) = \sqrt{\sec x}$
  9. Find the equation of the tangent line to the curve at the given point.
    1. $y = (1 + 2x)^{10}, \qquad (0, 1)$
    2. $y = \sin(\sin(x)), \qquad (\pi, 0)$

    10. The rest of these problems will appear on next week's homework. Skip the rest of these for Homework 5.

  11. Isolate $z$ in the equation \[-\sin(xy)(x + xz) = 1 + \cos(y)z\]
  12. Find $dy/dx$ with implicit differentiation for the equation \[x^3 + y^3 + x^2 + y = 1\]
  13. Consider the equation $x^{2/3} + y^{2/3} = 4$. Find the equation of the tangent line at the point $(-3\sqrt{3}, 1)$.
  14. Find $dy/dx$ by implicit differentiation. Assume $y$ is the dependent variable and $x$ is the independent variable.
    1. $x^2 - 4xy + y^2 = 4$
    2. $y \cos x = x^2 + y^2$
    3. $\cos(xy) = 1 + \sin y$
