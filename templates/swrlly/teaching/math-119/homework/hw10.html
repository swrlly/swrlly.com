Homework 10

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. For each of the following problems, find an equation of the hyperbola with vertex at the origin satisfying the following conditions or state it's impossible to.
    1. Foci $(\pm 5, 0)$, vertices $(\pm 3, 0)$
    2. Foci $(0, \pm \sqrt{10})$, vertices $(\pm 1, 0)$
    3. Asymptotes $y = \pm x$, vertices $(0, \pm 3)$
    4. Length of transverse axis is 6, oriented vertically.
      Hint: find the vertices.
    5. Foci $(\pm 5, 0)$, length of transverse axis is 6
    6. This hyperbola:
  2. Consider the hyperbola \[\dfrac{x^2}{4} - \dfrac{y^2}{8} = 1\] If I want to shift it three units right and four units down, what is the new formula?
  3. For the following ellipses, sketch the graph and find the foci.
    1. $\dfrac{x^2}{9} + \dfrac{(y+5)^2}{25} = 1$
    2. $\dfrac{(x + 1)^2}{36} + \dfrac{(y+1)^2}{64} = 1$
    3. $25(x-1)^2 + 9(y-2)^2 = 25 * 9$
  4. For the following parabolas, sketch the graph and find the focus.
    1. $(x-3)^2 = 8(y+1)$
    2. $(y+1)^2 = 16(x - 3)$
    3. $y^2 - 6y - 12x + 33 = 0$
    4. $x^2 + 2x - 20y + 41 = 0$
  5. For the following hyperbolas, sketch the graph and find the foci.
    1. $\dfrac{(x + 1)^2}{9} - \dfrac{(y-3)^2}{16} = 1$
    2. $x^2 - 2x + 1 - (y^2 - 2y + 1) = 1$
    3. $4(y-1)^2 - (x+1)^2 = 16$
