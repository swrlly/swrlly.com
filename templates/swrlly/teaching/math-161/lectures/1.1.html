1.1: Four Ways to Represent a Function

A function is a rule that assigns each element $x$ in a set $A$ with exactly one element $f(x)$ in another set $B$.

When you think of a function, consider the relationship between time and temperature at one physical location.

One single time $t$ will only be associated with one temperature $f(t)$; it cannot be two or more different temperatures.

Function Evaluation, Domain, and Graphs

Given $f(x) = x^2 + 2x + 3$, find
  1. $f(1)$
  2. $f(a)$
  3. $f(-a)$
  4. $f(x+h)$
Given \[f(x) = \dfrac{x+1}{x-1}\], find
  1. $f(1)$
  2. $f(x+h)$
The domain of a function $f$ is the set of all numbers you are allowed to plug in.
Find the domain of the following functions: \[f(x) = \dfrac{1}{x}\qquad g(x) = \dfrac{1}{x^2 - x} \qquad h(t) = \dfrac{1}{\sqrt{t}(t-1)}\]
The graph of a function $f$ is the set of points \[\{(x, f(x)) : x \text{ is in the domain of }f\}\]

Because $f(x)$ is the $y$ coordinate in the ordered pair, we will sometimes replace $F(x)$ with $y$.

Find the graph of $y = x^2$.

Piecewise Defined Functions

A piecewise function is a function that has different formulas on different parts of the domain.

Graph the function \[f(x) = \begin{cases}1 - x & x \leq - 1 \\ x^2 & x > -1\end{cases}\]
Graph the function $y = \lvert x \rvert$.

The graph helps us determine if a curve in the plane is actually a function.

Vertical Line Test
A curve in the $xy$-plane is the graph of a function of $x$ if and only if no vertical line intersects the curve more than once.
