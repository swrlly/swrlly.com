Homework 8

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. If \[f(x) = x^2 - 1 \qquad g(x) = x - 1\] find the function $\dfrac{f(x)}{g(x)}$ and state it's domain.
  2. Redefine these rational functions with their domain exclusions (also called holes)
    1. $f(x) = \dfrac{x^2 - 3x + 2}{x - 1}$
    2. $g(t) = \dfrac{2t^2 - 5t + 2}{2t - 1}$
    3. $f(x) = \dfrac{(x-3)(x^2 - 2x + 1)}{x-1}$
  3. Find the range of the following functions. Use transformations!
    1. $f(x) = \sqrt{x}$
    2. $f(x) = \sqrt{x - 1}$
    3. $f(x) = -\sqrt{x + 1}$
    4. $f(x) = 4 - x^2$
  4. Draw a graph of a one-to-one function.
  5. Suppose $f$ is not a one-to-one function. Explain what property is violated when we try to define the inverse $f^{-1}$.
  6. Determine if the following functions are inverses of each other using the Inverse Function Property.
    1. $f(x) = 2 - 5x, \ g(x) = \dfrac{2 - x}{5}$
    2. $f(x) = x^2, \ g(x) = \sqrt{x}$
    3. $f(x) = \dfrac{1}{x - 1}, \ g(x) = \dfrac{1}{x} + 1$
    4. $f(x) = x + 1, \ g(x) = \dfrac{1}{x - 1}$
  7. The function $f(x) = (x-1)^2$ is not one-to-one. What could we restrict the domain to to make $f(x)$ one-to-one?
  8. Given this graph of a function: find
    1. $f^{-1}(2)$
    2. $f^{-1}(5)$
    3. $f^{-1}(6)$
  9. If $f(1) = 2, f(2) = 3$ and $f(3) = 1$ find $f^{-1}(1), f^{-1}(2), f^{-1}(3)$.
  10. Find the inverse function of each of the following functions:
    1. $f(x) = 3x + 5$
    2. $f(x) = -2x - 4$
    3. $f(x) = \dfrac{1}{x + 2}$
    4. $f(x) = \dfrac{2x + 5}{x - 7}$
    5. $f(x) = 4-x^2, \qquad x \geq 0$
    6. $f(x) = \sqrt{4 - x^2}, \qquad 0 \leq x \leq 2$
    7. $f(x) = \dfrac{1}{x^2}, \qquad x > 0$
  11. Why is it helpful to put a quadratic into standard form?
  12. A quadratic $f(x) = x^2 + bx$ is given. What do you need to add and subtract in order to complete the square?
  13. A quadratic function $f(x)$ is given. Do the following four tasks for each function:
    • Express $f(x)$ in standard form.
    • Sketch a graph of $f(x)$.
    • (skip this) Find the maximum or minimum value of $f(x)$.
    • (skip this) Describe the type of solutions of $f(x)$ (use the discriminant).
    1. $x^2 - 2x + 3$
    2. $-x^2 + 6x + 4$
      Rewrite $-x^2 + 6x + 4 = -x^2 - (-6)x + 4 = -(x^2 - 6x) + 4$ and complete the square.
    3. $-3x^2 + 6x - 2$
    4. $2x^2 + 12x + 10$
    5. $x^2 + 2x$
    6. $x^2 + 4x - 1$
