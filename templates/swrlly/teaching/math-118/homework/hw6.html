Homework 6

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. Suppose \[f(x) = x \qquad g(x) = \dfrac{1}{x-2} \qquad h(x) = \sqrt{x} \qquad i(x) = \dfrac{1}{x-3}\]
    1. Find $f(x) + g(x) - h(x) - i(x)$ and it's domain.
    2. Find $f(x) \cdot g(x) \cdot h(x) \cdot i(x)$ and it's domain.
  2. Suppose \[f(x) = x^2 \qquad g(x) = \sqrt{x} \qquad h(x) = \dfrac{1}{x - 1} \qquad i(x) = \dfrac{1}{x-2}\]
    1. Find $f(x) + g(x) - h(x) - i(x)$ and it's domain.
    2. Find $\dfrac{f(x) \cdot g(x) \cdot h(x)}{i(x)}$ and it's domain.
  3. If $f(x) = x^2 + 1$ and $g(x) = x - x^2$, find $f(x) - g(x)$ and $f(x)g(x)$.
  4. Given the functions $f(x) = \sqrt{x - 2}$ and $g(x) = 5 - x^2$, find the following and fully simplify:
    1. $f(g(0))$
    2. $f\circ g$
    3. $g \circ f$
    4. $f \circ f$
    5. $g \circ g$
  5. Given the following functions $F(x)$, find two functions $f$ and $g$ where $f\circ g = F$. You are not allowed to choose $f(x) = x$ or $g(x) = x$.
    1. $F(x) = (x-3)^{-\frac{1}{2}}$
    2. $F(x) = \sqrt{x - 2} + \dfrac{1}{\sqrt{x - 2}}$
    3. $F(x) = \sqrt{(x^2 + 2x + 3)^3}$
  6. Draw a graph of a one-to-one function.
  7. Suppose $f$ is not a one-to-one function. Explain what property is violated when we try to define $f^{-1}$.
  8. Suppose $f(x) = 2-5x$ and $g(x) = \dfrac{2 - x}{5}$. Are $f(x)$ and $g(x)$ inverses of each other? Show using the Inverse Function Property.
  9. Suppose $f(x) = x^2$ and $g(x) = \sqrt{x}$. Are $f(x)$ and $g(x)$ inverses of each other? Show using the Inverse Function Property.
  10. The function $f(x) = x^2$ is not one-to-one. What could we restrict the domain to to make $f(x)$ one-to-one?
  11. Find the inverse function of each of the following functions:
    1. $f(x) = 3x + 5$
    2. $f(x) = \dfrac{1}{x + 2}$
    3. $f(x) = 4-x^2, \qquad x \geq 0$
    4. $f(x) = \sqrt{4 - x^2}, 0 \leq x \leq 2$
    5. $f(x) = \dfrac{1}{x^2}, \qquad x > 0$
  12. Given this graph of a function: find
    1. $f^{-1}(2)$
    2. $f^{-1}(5)$
    3. $f^{-1}(6)$
