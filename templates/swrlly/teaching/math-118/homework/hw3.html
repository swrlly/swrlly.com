Homework 3

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. Perform the indicated operation and/or fully simplify. Get rid of all negative exponents.
    1. $\dfrac{1}{x + 1} - \dfrac{1}{x - 1}$
    2. $\dfrac{1 + \dfrac{1}{x}}{\dfrac{1}{x} - 2}$
    3. $\dfrac{x}{x + 3} - \dfrac{18}{x^2 - 9}$
    4. $\dfrac{1}{x-1} + \dfrac{1}{x} - \dfrac{1}{x-1}$
    5. $\dfrac{x^{-1} + y^{-1}}{4} \ \ $ (hint: use definition of negative exponent. This is a compound fraction)
    6. $\dfrac{\dfrac{1}{\sqrt{x+h}} - \dfrac{1}{\sqrt{x}}}{h} \ \ $ (hint: get rid of the compound fraction, rationalize the numerator, and cancel common factors)
  2. A student tries to simplify \[\dfrac{x^{-1} + y^{-1}}{4} = \dfrac{1}{4xy}\] Which exponent law was violated?
  3. A student tries to simplify \[\dfrac{x+3}{x + 2} \cdot 4 = \dfrac{x+3\cdot 4}{x+2} = \dfrac{x+12}{x+2}\] What mistake did the student make?
  4. Show why $x = 2$ is a solution to the equation \[\dfrac{1}{x} - \dfrac{1}{x-4} = 1\]
  5. Rationalize the denominator in $\dfrac{3}{\sqrt{x} - \sqrt{y}}$
  6. Isolate the given variable in the following equations:
    1. $4x + 2 = 6x - w, \ $ for $x$
    2. $\dfrac{x + 1}{x - 3} = 1, \ $ for $x$
    3. $4xy - w(2xz - 3yz) + 3 = -4x - y, \ $ for $x$
  7. Find all real-valued solutions for the following equations.
    1. $3x + 4 = 7$
    2. $2x + 3 = 7 - 3x$
    3. $x^2 + x - 12 =0$
    4. $2x^2 = 8$
    5. $\dfrac{1}{x} = \dfrac{4}{3x} + 1$
    6. $\dfrac{1}{x-1} + \dfrac{1}{x+2} = \dfrac{5}{4}$
  8. Isolate $x$ in the equation $a(b + cx) + d = e$. Simplify any compound fractions.
  9. A student tries to isolate $x$ in the equation $(a + b)x = c + d$ by dividing by $a + b$, resulting in \[x = \dfrac{c}{a + b} + d\] What mistake did the student make?
  10. Solve the following equations. Remember to check your work if necessary!
    1. $x^2 + x - 12 = 0$
    2. $4x^2 - x = 0$
    3. $x^2 = 3(x - 1)$
    4. $\dfrac{1}{x-1} - \dfrac{2}{x^2} = 0$
    5. $\sqrt{8x - 1} = 3$
    6. $\sqrt{2x + 1} + 1 = x$

    11. The rest of these problems will be on next week's homework. Skip for now.

  11. Add, subtract, divide or multiply. Make sure your answer is in the form $a + bi$.
    1. $(3 + 2i) + 5i$
    2. $(-12 + 8i) - (7 + 4i)$
    3. $(5 - 3i)(1 + i)$
    4. $(3 - 7i)^2$
    5. $\dfrac{1}{i}$
    6. $\dfrac{2 - 3i}{1 - 2i}$
    7. $i^{10}$
  12. Solve $3x^2 + 1 = 0$. Make sure your solution is a complex number.
