Homework 10

Directions:

  1. Show each step of your work and fully simplify each expression.
  2. Turn in your answers in class on a physical piece of paper.
  3. Staple multiple sheets together.
  4. Feel free to use Desmos for graphing.

Answer the following:

  1. Express the equation in exponential form:
    1. $\log_35 = x$
    2. $\log_381 = 4$
    3. $\ln(x-1) = 4$
    4. $\ln(x^2 - x - 2) = 3$
  2. Evaluate the expression:
    1. $\log_22$
    2. $\log_636$
    3. $\log_{10}\sqrt{10}$
    4. $\log_{49}7$
    5. $\log_77^{10}$
    6. $\ln(1/e)$
    7. $e^{\ln3}$
    8. $\ln e^4$
    9. $\log_51$
  3. Use the definition of the logarithm to find $x$.
    1. $\log_3 x = -2$
    2. $\log_x16 = 4$
    3. $\ln e^2 = x$
    4. $\log_x6 = \dfrac{1}{2}$
    5. $\ln x = -1$
  4. Explain why $f(x) = \ln(x - 3)$ is horizontally shifted to the left three units from $g(x) = \ln x$.
  5. Use transformations to sketch the following:
    1. $f(x) = \ln(-x)$
    2. $f(x) = - \ln(- x - 2)$
  6. State which of the following are true or false:
    1. $\ln 3x = \ln 3 + \ln x$
    2. $\ln(3 + x) = \ln 3 + \ln x$
    3. $\ln(x(x-1)) = \ln x + \ln (x - 1)$
    4. $\ln x^2 = 2\ln x$
    5. $(\ln x)^2 = 2\ln x$
    6. $\log_2(x - 3) = \log_2 x - \log_2 3$
    7. $\ln\left(\dfrac{x}{x - 1}\right) = \ln x - \ln (x - 1)$
  7. Use Laws of Logarithms to evaluate the expression without a calculator:
    1. $\log 5 + \log 20$
    2. $\log_5 \sqrt{5}$
    3. $\log_2 60 - \log_2 15$
    4. $\log_4 16^{100}$
    5. $\ln (\ln e^{e^{200}})$
  8. Expand these logarithms using Laws of Logarithms:
    1. $\log_3 4x^2$
    2. $\ln \sqrt{ab}$
    3. $\ln \dfrac{3x^2}{(x - 1)^4}$
    4. $\ln \sqrt[3]{\dfrac{a}{b + 2}}$
    5. $\log_7 ab^2$
    6. $\ln(\ln(x)\ln(y))$
  9. Find the solution of the exponential equation. Write your answer as a number unless the logarithm is not easy to evaluate.
    1. $5^{x - 1} = 125$
    2. $5^{2x-3} = 1$
    3. $e^{1-4x} = 2$
    4. $5^x = 4^{x+1}$
    5. $e^{2x} - 3e^x + 2 = 0$
    6. $x^22^x - 2^x = 0$
    7. $x^2e^x + xe^x - e^x = 0$
  10. Find the solution of the logarithmic equation. Write your answer as a number unless the exponent is not easy to evaluate.
    1. $\log x + \log(x - 1) = \log(4x)$
    2. $\ln x = 10$
    3. $\ln(2 + x) = 1$
    4. $2\log x = \log 2 + \log(3x - 4)$
    5. $\log x + \log(x - 3) = 1$
    6. $\log_2(x^2 - x - 2) = 2$
    7. $\log_2(\log_3 x) = 4$
  11. How long will it take for an investment of $1000 to double in value if the interest rate is 8.5% per year, compounded continuously?
  12. You invest $6500 in an account that pays 6% interest per year, compounded continuously.
    1. What is the APY?
    2. How long will it take for your investment to double?
